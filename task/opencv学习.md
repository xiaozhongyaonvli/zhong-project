**其他处理噪声** 
* 中值滤波 (Median Filter)：
中值滤波是一种非线性滤波方法，通过将像素值替换为其邻域像素值的中值来减少噪声，特别适用于去除椒盐噪声。
* 双边滤波 (Bilateral Filter)：
双边滤波在保留边缘的同时进行平滑处理。它结合了空间域和色彩域的加权平均。
* 非局部均值滤波 (Non-Local Means Denoising)：
非局部均值滤波通过考虑相似的图像块来减少噪声，效果非常好，但计算量较大。
* 均值滤波 (Mean Filter)：
通过简单地对邻域像素值进行平均来减少噪声。尽管会模糊图像细节，但实现简单。
**椒盐噪声**
一种常见的图像噪声类型，其特点是图像中出现随机分布的黑色和白色噪点，类似于盐和胡椒的颗粒。这种噪声在图像中呈现为随机的亮点（盐）和暗点（胡椒）
椒盐噪声通常由以下几种原因引起：
* 传感器故障：相机传感器出现故障或缺陷。
* 传输错误：在图像传输过程中，数据丢失或错误。
* 压缩算法：某些图像压缩算法可能会引入椒盐噪声。

## 边缘检测

### 基本概念
#### 边缘：边缘是图像中亮度急剧变化的区域。通常是物体的轮廓、边界或者其他显著的结构。边缘检测的目标是识别这些亮度变化的区域。
#### 梯度：边缘通常与图像中灰度值的梯度有关。梯度是图像中灰度值变化的速率，通常用来衡量图像中每一点的边缘强度和方向。
Sobel 算子: 计算图像中每个点的梯度幅度和方向，适合于检测边缘。
Prewitt 算子: 类似于 Sobel 算子，但权重分布不同，效果类似。
Scharr 算子: 更高精度的梯度计算，响应强，对边缘的检测更敏感。
Laplacian 算子: 计算二阶导数，适合于边缘检测和噪声处理。
#### 非最大抑制:在边缘检测中，非最大抑制用于细化边缘，通过抑制边缘附近的非极大值，确保边缘的清晰度。
1. 计算梯度强度和方向:使用 Sobel 算子计算图像的梯度强度和方向。初始化输出图像:创建一个与梯度强度图大小相同的输出图像，用于存储非最大抑制后的结果。
2. 遍历每个像素点:对于每个像素点，确定其梯度方向，并根据该方向在梯度强度图中查找相邻的两个像素点。
3. 比较强度值:
比较当前像素点的梯度强度值与其在梯度方向上的两个相邻像素点的强度值。
如果当前像素点的强度值是最大值，则保留该像素点的强度值；否则，将其强度值设为零。
4. 输出结果:最终的输出图像即为经过非最大抑制后的结果，其中只有局部极大值被保留，其他被抑制为零。
用于在边缘检测和目标检测中进行局部极大值的提取。其目的是在边缘图像中找到最突出的边缘点，或者在目标检测中抑制重复的检测框，仅保留最具代表性的检测框。
* 滞后阈值:滞后阈值用于连接边缘，确保在检测到的边缘之间有连续性。通过设置高阈值和低阈值来分离真实边缘和噪声
高阈值（High Threshold）: 用于确定一个像素是否是边缘的强证据。如果像素的强度值高于这个阈值，那么这个像素被认为是边缘的一部分。
低阈值（Low Threshold）: 用于确定一个像素是否是边缘的一部分，基于它是否与已经标记为边缘的像素连接。如果像素的强度值低于这个阈值但与强边缘连接，那么这个像素也被认为是边缘的一部分。
滞后阀值的作用
减少噪声: 通过高阈值和低阈值的双重检查，可以有效减少噪声对边缘检测结果的影响。
连接断裂边缘: 允许连接不连续的边缘，使得检测结果更加连贯和完整。
提高边缘检测的鲁棒性: 在实际应用中，边缘检测常受到噪声和不连续性的影响，滞后阀值可以增强边缘检测的鲁棒性。

### 常用边缘检测算法
#### 知识前瞻--卷积 (一般用于模糊，锐化，边缘处理)
卷积核（Filter/Kernal）:
卷积核是一个小的矩阵，通常比输入图像的尺寸小得多。例如，一个常见的卷积核是 3×3 的矩阵。
每个卷积核中的值是权重，用于确定滤波器的行为。
卷积操作:
卷积操作将卷积核应用于图像的每个像素位置。对于图像中的每个位置，卷积核与该位置上的像素及其邻域进行逐元素相乘，然后将结果求和，得到输出图像中的一个像素值。
卷积核在图像上滑动（步幅），并在每个位置执行上述操作。
边界处理:
当卷积核滑动到图像边缘时，可能会遇到没有足够像素的数据。边界处理方法决定了如何处理这些情况，包括常数填充、镜像填充等。
#### Canny边缘检测算法
经典且广泛使用的边缘检测算法，它是一种多阶段的算法，旨在检测图像中真正的边缘，同时减少噪声的影响。
**步骤**
1. 高斯滤波
2. 计算梯度
3. 非最大抑制
4. 滞后阀值
```
 // 高斯平滑
    cv::Mat smoothed_image;
    cv::GaussianBlur(image, smoothed_image, cv::Size(5, 5), 1.5);

    // Canny 边缘检测
    cv::Mat edges;
    cv::Canny(smoothed_image, edges, 100, 200);
```
高斯滤波器的大小和标准差：影响图像平滑的程度，需要根据图像和噪声情况调整。
Canny 函数中的阈值：影响边缘检测的灵敏度，通常需要通过实验确定合适的值。

## 膨胀腐蚀操作
* 腐蚀（Erosion）
腐蚀操作会减少图像中的白色区域。它的基本原理是通过一个结构元素（kernel）扫描图像，只有当结构元素完全包含在图像的白色区域时，中心点才会被保留，否则它会被移除。腐蚀操作可以去除小的噪声，使物体的边界收缩，分离相连的物体。
* 膨胀（Dilation）
膨胀操作会增加图像中的白色区域。它的基本原理是通过一个结构元素（kernel）扫描图像，只要结构元素和图像的白色区域有重叠，中心点就会被保留。膨胀操作可以填补图像中的小孔，使物体的边界扩展，连接相邻的物体。
结构元素（Kernel）是一个小矩阵，它决定了膨胀和腐蚀的具体方式。常见的结构元素形状有矩形、椭圆形和十字形。 
cv::Mat cv::getStructuringElement(int shape, cv::Size ksize, cv::Point anchor = cv::Point(-1, -1))
shape: 矩形（MORPH_RECT）、椭圆形（MORPH_ELLIPSE）、十字形（MORPH_CROSS）
ksize: 指定结构体大小
anchor: 锚点，这是一个 cv::Point 类型的参数，指定了结构元素的锚点（即结构元素的中心点）。默认值是 cv::Point(-1, -1)，表示结构元素的中心点将自动设为元素的中心。
一般情况下，你可以使用默认值，除非你有特定的需求。
eg: cv::Mat rect_kernel = cv::getStructuringElement(cv::MORPH_RECT, cv::Size(5, 5));

最后的参数是运行次数。

首先对图像进行一次膨胀操作，然后进行一次腐蚀操作。这个组合称为 开运算（Opening Operation）。开运算的作用是去除小的噪点，同时保持图像中较大的白色区域的形状。具体来说：
膨胀操作会扩展白色区域，填补图像中的小黑点。
随后的腐蚀操作会收缩扩展后的白色区域，恢复白色区域的原始大小，但已经去除了小的噪点。

## 霍夫变换
霍夫变换（Hough Transform）是一种用于检测图像中几何形状（如直线、圆等）的技术。最常用的是直线霍夫变换
### 基本原理
霍夫变换将图像中的点（通常是边缘点）映射到一个参数空间。
对于直线检测，每个点在参数空间中会变成一个曲线（在二维情况下是一个正弦曲线）。
曲线交点的聚集位置代表图像中存在的直线。这是因为不同的边缘点可以通过相同的直线进行连接，因此它们的映射会在参数空间中汇聚成点。
### 步骤
边缘检测：首先需要使用边缘检测算法（如Canny边缘检测）来找到图像中的边缘。
霍夫变换：在参数空间中检测这些边缘点的曲线交点。
参数空间：通常使用极坐标（rho, theta）表示直线，其中 rho 是直线到原点的距离，theta 是直线的角度。
eg:
std::vector<cv::Vec4i> lines;
cv::HoughLinesP(eroded_edges, lines, 1, CV_PI / 180, 50, 50, 10);
cv::HoughLinesP 是用于检测图像中的直线的霍夫变换函数。
eroded_edges 是经过边缘检测和形态学操作后的图像。
lines 是存储检测到的直线的容器。
参数说明：
1 是距离精度（以像素为单位）。
CV_PI / 180 是角度精度（即θ的步进，单位为弧度）。
50 是累加器阈值。只有当累加器的值超过此阈值时，才会认为存在直线。 增大累加器阈值，减少检测到的线条数量
50 是最小直线长度，只有长度大于这个值的线段才会被保留。 增大最小直线长度，过滤掉短线段
10 是线段之间的最小间距，确保检测到的线段不太靠近。 增大最小线段间距，减少线条重叠


# 专业术语
## 鲁棒(稳定性)
“鲁棒”是一个技术术语，来源于英文“robust”，意思是系统、算法或方法在面对变化或不确定性时仍然能够表现出稳定和有效的性能。换句话说，一个鲁棒的系统或算法能够在多种情况下，包括预期的和未预期的情况下，都能保持其功能和性能。
在图像处理和计算机视觉中，鲁棒性意味着算法能够应对各种干扰和变化，例如：
光照变化：算法在不同光照条件下仍能工作。
噪声：图像中存在噪声时，算法仍能识别目标。
尺度变化：目标物体的大小发生变化时，算法仍能检测到目标。
旋转和翻转：目标物体发生旋转或翻转时，算法仍能正确识别。
遮挡：目标物体部分被遮挡时，算法仍能识别出目标。
